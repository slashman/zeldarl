# Running using Swing

You can edit font.sz with the name of a mono-spaced font installed in your system. 

# Running using JCurses

In addition to the emulated terminal window, the game can run in a real terminal mode via JCurses.

For windows, you can use the provided bat file, zrl-jcurses.bat

For linux and mac, make sure to provision a valid libjcurses.so in the \lib directory, then run

java -jar zrl.jar jc
